## Machine Learning Project
Hotel Booking Demand: Classification and Prediction of Cancellations

---

### :bar_chart: Project Description
This project, developed as part of the **Machine Learning** course within the MSc in Data Science, focuses on building and evaluating multiple classification models for predicting customer behaviour in terms of **hotel booking cancellations**.  

The entire pipeline was implemented using **KNIME Analytics Platform**. 

---

### :dart: Objectives
- Explore and preprocess **hotel booking data**
- Perform **feature engineering** and **feature selection**
- Train and compare multiple **predictive models** using appropriate **metrics**
- Interpret results and extract **actionable insights**

---

### :file_folder: Repository Structure
- `report.pdf` :arrow_right: exam report
- `workflow.knwf` :arrow_right: KNIME workflow implementing the full ML pipeline

---

### :arrow_forward: Project Execution
1) Download the dataset from Kaggle (https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand)
